RVV benchmark Tenstorrent Tensix T2 (Tenstorrent Blackhole)

The Tenstorrent Blackhole is an AI accelerator built out of Tensix tiles, which each contain 5 32-bit "baby" RISC-V cores, one of which supports a subset of Zve32f. The Zve32f subest doesn't support the vdiv/vdivu/vrem/vremu/vfdiv/vrfdiv/vfsqrt/vfrsqrt7/vfrec7 instructions, the vcsr CSR and doesn't have compliant floating-point semantics for denormals, rounding, and FMA. The cores runs at 1.35 GHz and have a vector length of 128-bit. See the Tenstorrent documentation for more detail.
Blackhole separately also contains 16 Linux capable SiFive X280 cores, see the benchmarks for the X280.

Instruction timings

The following are measured cycle averages when unrolling and looping over the given instruction, which should estimate the instruction throughput. The registers involved are randomized to usual values, that is floating point values are real numbers and not NaN/Inf, and instructions like vslide* stay within the vl range.
