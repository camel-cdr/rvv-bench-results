RVV benchmark MegaBobcat

Bobcat is Tenstorrents open source fork of BOOM, which adds vector extension support. It supports a large subset of the ratified 1.0 version of the V vector extension including half precision floating points, and has a VLEN of 256 bits. The unimplemented instructions are: vdiv, vdivu, vfsqrt, vfsqrt7, vfrec7, vfdiv, vfrdiv, vremu, vrem.
All measurements were done on rtl simulation of the master branch on 2024-07-18.
