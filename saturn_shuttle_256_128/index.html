RVV benchmark Saturn

Saturn is a parameterized open-source RVV 1.0 vector unit integrated into the Chipyard framework. It supports different VLEN and DLEN configurations, as well as "full chaining with zero dead-time". This benchmark was done on the VLEN=256 DLEN=128 configuration connected to a shuttle core, which is a two-issue core based on rocket-chip (GENV256D128ShuttleConfig).
All measurements were done on rtl simulation of the master branch from 2024-08-18.

Instruction timings

The following are measured cycle averages when unrolling and looping over the given instruction, which should estimate the instruction throughput. The registers involved are randomized to usual values, that is floating point values are real numbers and not NaN/Inf, and instructions like vslide* stay within the vl range.
